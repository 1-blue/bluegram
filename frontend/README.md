## 1. bluegram - SNS 클론
`CRA`를 사용하지 않고 직접 기본 설정 구현

## 2. 개발할 기능들
+ 버전 1.0.0
	1. 회원가입, 로그인
	2. 게시글 CRUD
	3. 댓글 CRUD
	4. 답글 CRUD
	5. 내 정보 CRUD
	6. 본인 게시글 검색
	7. 좋아요 게시글 검색
	8. 해시태그
	9. 팔로잉, 팔로워
	10. OAuth ( 네이버, 구글, 카카오 로그인 )
